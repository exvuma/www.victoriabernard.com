== Description ==

Elegant and clean, Simona is a theme designed to display your special day with stunning beauty. With a fashionable clean look, Simona is a wedding theme that takes just minutes to setup. The theme fits for Engagement, Wedding Event, Wedding Planner, Video Maker, Wedding Photography, Flower Designer and any kind of Wedding Industries and activities.

== Installation ==
	
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
4. If you're replacing your old theme, we recommend using a plugin to regenerate the image thumbnails like Regenerate Thumbnails (https://wordpress.org/plugins/regenerate-thumbnails/).

== Header Slider Setup ==

1. In your admin panel, go to Appearance > Customize > Header Slider.
2. Add the images that you want ( Simona Theme recommends a size of 1600 × 800 pixels. ).
3. Click on 'Save & Publish' button.
